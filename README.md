# semog.nl

A polished blog starter built with [Zola](https://www.getzola.org/), using the [Terminus](https://www.getzola.org/themes/terminus/) theme, and ready for deployment to GitHub Pages at `https://semog.nl`.

## Local development

```bash
zola serve
```

## Build

```bash
zola build
```

## Deploy

Push to `main` and let GitHub Actions build and publish the site.

## Theme

The `Terminus` theme is vendored in `themes/terminus` so the project works without setting up git submodules.
